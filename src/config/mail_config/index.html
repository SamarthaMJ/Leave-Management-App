Mail Configuration

The following code configures the mail connection using FastAPI Mail based on the provided settings. It checks if the necessary MAIL_USERNAME and MAIL_PASSWORD values are set, and, if so, initializes the mail configuration.

Attributes

MAIL_USERNAME

  • Type: str
  • Description: The username for the mail server, usually an email address.

MAIL_PASSWORD

  • Type: str
  • Description: The password associated with the mail server account.

MAIL_FROM

  • Type: str
  • Description: The email address used as the sender.

MAIL_PORT

  • Type: int
  • Description: The port used by the mail server.

MAIL_SERVER

  • Type: str
  • Description: The hostname of the mail server.

MAIL_FROM_NAME

  • Type: str
  • Description: The name to display for the sender in the email.

MAIL_STARTTLS

  • Type: bool
  • Description: Specifies whether to use STARTTLS for secure connection.

MAIL_SSL_TLS

  • Type: bool
  • Description: Specifies whether to use SSL/TLS for the connection (set to False to disable).

USE_CREDENTIALS

  • Type: bool
  • Description: Whether to use credentials for connecting to the mail server.

VALIDATE_CERTS

  • Type: bool
  • Description: Whether to validate the mail server's SSL/TLS certificates.

TEMPLATE_FOLDER

  • Type: str
  • Description: The path to the folder containing email templates.

Conditional Initialization

If both MAIL_USERNAME and MAIL_PASSWORD are provided in the settings, the ConnectionConfig is initialized with the relevant configuration values. If either of the two values is missing, the mail configuration will be set to None.
