Token Model

Overview

The token.py file defines Pydantic models for authentication and token management.

Models

Token

Authentication token details: - access_token - token_type - Associated user information

TokenDecrypted

Detailed token payload: - User context information - Designation - Role details - Factory and line information - Authentication scopes - Expiration timestamp
